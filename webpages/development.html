This is v.1.0 of my portfolio project. I will be documenting my progress and implementations along the way. This version uses pure html5, css, and javascript. The first iteration will be playing with the classic design rules of contrast with an attempt to defy some design formalities. In this example, h1 should be bolder than the subheadings. Though my aesthetic decisions seem a tad bit questionable, I am illustrating that you can push the boundaries of aesthetics playing with minute details.

Version 1.0 - Static development with Vanilla functionalities

Last implemented on Sunday, February 18th, 2024 Features:
  1. Flexbox Layout: Offers dynamic window and element resizing
  2. Animation: Elements fade-in using built-in css cascading animations
  3. Night Mode Toggle: toggle in and out of display themes manually
